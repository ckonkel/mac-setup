## Requirements

Ansible installed
https://docs.ansible.com/ansible/latest/installation_guide/intro_installation.html#latest-releases-via-pip

Run in terminal:

* `$ sudo easy_install pip`
* `$ sudo pip install ansible`

## Install

Run in terminal:

* `ansible-playbook main.yml`
